## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Serve production files using static server
```
npm run serve-static
```

### Lints and fixes files
```
npm run lint
```

### Deploy using firebase
```
npm run deploy
```
